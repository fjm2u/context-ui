# @context_ui/core

Core schema/spec helpers plus the registry-first ContextUI component.

## Docs
- Package docs: `../../docs/core.md`
- Docs index: `../../docs/index.md`
- Example app: `../../examples/react-app`

## Commands
- Build: `pnpm --filter @context_ui/core build`
- Tests: `pnpm --filter @context_ui/core test`

## Notes
`@context_ui/core` ships without built-in components. Import `@context_ui/theme` for base styles and provide your own registry.
