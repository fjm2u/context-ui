# @context_ui/theme

Default Tailwind styles for ContextUI. **WIP**: This package is under active development; the API/docs may change.

## Docs
- Package docs: `../../docs/theme.md`
- Docs index: `../../docs/index.md`

## Commands
- Build: `pnpm --filter @context_ui/theme build`
- Tests: `pnpm --filter @context_ui/theme test` (none yet)

## Notes
Importing `@context_ui/theme` applies base styles but does not register any components.
